# Funding Data

This directory contains JSON dumps and structured data from various U.S. funding sources:

- **SBIR**: Small Business Innovation Research awards
- **NSF**: National Science Foundation grants
- **NIH RePORTER**: NIH research project data
- **DIU**: Defense Innovation Unit contracts
- **In-Q-Tel**: Investment data
