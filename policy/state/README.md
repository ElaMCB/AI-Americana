# State Policy Documents

This directory contains state-level AI policy documents organized by state abbreviation.

## Current States

- **CA**: California (e.g., SB-1001)
- **TX**: Texas (e.g., HB-2273)
- **NY**: New York
- **DC**: District of Columbia

## Format

Each state subdirectory should contain markdown/text copies of bills, regulations, and other policy documents related to AI.
